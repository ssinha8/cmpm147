Description

In this assignment, we designed a generator that takes an existing image and renders shapes in an attempt to form a version of the original image. It uses it's own evaluation score to try and recreate the original image using the generated shapes.

Technical

  1. Using the dropwdown key, you can find three seperate images that the generator attempts to replicate.
  2. Each one will result in a completely different attempt from the generator to replicate the image.
  3. Using the 30 second annealing trick, (hopefully) you can see a design similar to the image in question.
  4. The mutation algorithm provided by Wes Modes makes incremental design changes based on the mutation rate. So if you slide the slider down to 0, the exploration should almost completely stop.
