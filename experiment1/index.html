Experiment 1 - Text Generator

Translated from the text generator made on glitch!

Description

The goal of this experiment was to create a text generator through Glitch and then move the generator onto my design website.

Technical

Working off of given source code, I rewrote my own unique story for a text generator. I then replaced the string tokens with my own token names and values. The generator provided by the given source code randomly assigns the tokens any of the associated values.

Reflection

I found this to be a success in introducing me to Glitch as well as some basic HTML review.
